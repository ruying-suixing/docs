# 快速了解

## 域名更换历程

最初（$2025/12$），我用 [Gridea](https://open.gridea.dev) 整了一个图形化界面的博客，本质上 Gridea 也是一个 SSG，那是，我怀着激动的心情~~没用梯子~~，用 [Outlook 邮箱](mailto:haoxuanzhang2014@outlook.com)注册了一个 [Github](https://github.com) 账号（那时似乎不用验证码），添加了 [Github Page](https://github.io)，还配置了 Github Token，此时，我用的唯一域名（无子域）是：[ruying-suixing.github.io](https://ruying-suixing.github.io)。

后来（$2026/02$ 末），我注册了一个免费的 <dpdns.org> 二级域名，用 [Cloudflare](https://cloudflare.com) 托管，添加了 CNAME 成功后，高兴得不得了，还写了一篇[文章](https://life.rusin.lol/post/dpdns-org/)来分享，迁移到 [Hexo](https://hexo.io) 后还更新了一下[新版](https://blog.rusin.lol/article/4620975/)；我同学[赵欣泽](https://luogu.com.cn/user/1217878)送了我免费的 [论坛](https://zjsh.freeflarum.com)。我还折腾了 [Vitepress](https://vitepress.dev) 文档，在infinityfree 上把**免费**虚拟主机当 html 托管+免费二级域名用，部署在 [ruyingsuixing.wuaze.com](https://ruyingsuixing.wuaze.com "下载因为法律原因，wuaze.com 似乎被下架了")那时，我用的唯三域名（无子域）是：[ruyingsuixing.dpdns.org](https://ruyingsuixing.dpdns.org)，[ruyingsuixing.wuaze.com](https://ruyingsuixing.wuaze.com "下载因为法律原因，wuaze.com 似乎被下架了")和 [Flarum](https://zjsh.freeflarum.com)。

再后来，我学会了使用子域名，于 $2026/03/19$ 将网站框架迁移到了 [Hexo](https://hexo.io)，用 [Anzhiyu 主题](https://docs.anheyu.com)，[Gridea](https://open.gridea.dev) 停止维护，在 [Dnshe](https://dnshe.com) 上整了个 cn.mt 后缀，迁移至 [life.rusin.cn.mt](https://life.rusin.cn.mt)。又学 [梦爱吃鱼](https://bsgun.cn) 做了主页，整了很多服务，并做了一周的 html 重定向到新域名，才把老域名删了。那时，因为 cn.mt 不在 PSL 里，我又接连迁移到 [rusin.dpdns.org](https://rusin.dpdns.org) 和 [rusin.qzz.io](https://rusin.qzz.io)，最终于 $2026/5/1x$ 迁移到 [rusin.lol](https://rusin.lol)，是在 [Namesilo](https://namesilo.com) 上买的。
