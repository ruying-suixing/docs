域名更换历程 ​

最初（2025/12），我用 Gridea 整了一个图形化界面的博客，本质上 Gridea 也是一个 SSG，那是，我怀着激动的心情没用梯子，用 Outlook 邮箱注册了一个 Github 账号（那时似乎不用验证码），添加了 Github Page，还配置了 Github Token，此时，我用的唯一域名（无子域）是：ruying-suixing.github.io。

后来（2026/02 末），我注册了一个免费的 <dpdns.org> 二级域名，用 Cloudflare 托管，添加了 CNAME 成功后，高兴得不得了，还写了一篇文章来分享，迁移到 Hexo 后还更新了一下新版；我同学赵欣泽送了我免费的 论坛。我还折腾了 Vitepress 文档，在infinityfree 上把免费虚拟主机当 html 托管+免费二级域名用，部署在 ruyingsuixing.wuaze.com那时，我用的唯三域名（无子域）是：ruyingsuixing.dpdns.org，ruyingsuixing.wuaze.com和 Flarum。

再后来，我学会了使用子域名，于 2026/03/19 将网站框架迁移到了 Hexo，用 Anzhiyu 主题，Gridea 停止维护，在 Dnshe 上整了个 cn.mt 后缀，迁移至 life.rusin.cn.mt。又学 梦爱吃鱼 做了主页，整了很多服务，并做了一周的 html 重定向到新域名，才把老域名删了。那时，因为 cn.mt 不在 PSL 里，我又接连迁移到 rusin.dpdns.org 和 rusin.qzz.io，最终于 2026/5/1x 迁移到 rusin.lol，是在 Namesilo 上买的。
